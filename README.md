# Utility-Tool Web App

A collection of simple utility tools built into a single website. I though of some easy useful features i use pretty often and I put them all in one place with a minimalist design.

**Test it out under: https://utilies-tool.vercel.app  or  https://manager-factice.vercel.app**

### Features
**1. Category - Chance and Decisions**
  - Random Number: Enter two numbers at you will get a random number in between them!
  - Spin the Wheel: A "wheel" where you can type in your own options (like names or food) and spin it to get a random winner!
  - Coin Flip: A simple instant heads or tails coin flipper!

**2. Category - Time and Progress**
  - Local Time: Just your local time in hours, minutes and seconds! (Also detects if you need 24h or 12h format)
  - World Clock: Just a clock for some time zones and their cities!
  - How Many Days Until?: A countdown to a specific date in days, hours and minutes!

  If you are on any if these sites, just click the Header to get backto the homepage!
